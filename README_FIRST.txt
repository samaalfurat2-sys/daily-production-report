Open INSTALLERS_FROM_GITHUB_AR_EN.md first.

Fast path:
1) Upload this project to GitHub.
2) Actions -> "Create Release with Download Links" -> Run workflow.
3) Go to Releases page -> download app-debug.apk, ProductionReportSetup.exe or windows-portable.zip.

Alternative (artifacts only, expire after 90 days):
Actions -> "Build Ready Installers" -> download windows-portable and android-apk-debug from Artifacts.
